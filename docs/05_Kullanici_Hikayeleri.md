# 05 - Kullanıcı Hikayeleri

Bu doküman, sistemin farklı kullanıcılar tarafından nasıl kullanılacağını kullanıcı hikayeleri formatında özetler.

## 1. Kargo Gönderme

**Hikaye 1:**  
Mağaza yöneticisi olarak, hazır siparişleri kolayca kargoya vermek istiyorum.  
**Kabul Kriterleri:**
- Sipariş listesinden tekli veya toplu “Kargoya Ver” butonu
- Kargo firması (Pusula Kargo, Meltem Dağıtım vb.) seçimi
- Otomatik kargo etiketi oluşturma

**Hikaye 2:**  
Operasyon ekibi olarak, aynı anda birden fazla siparişi kargoya verebilmek istiyorum.

## 2. Kargo Takibi

**Hikaye 3:**  
Kargo sorumlusu olarak, gönderilen tüm kargoların durumunu tek ekrandan takip etmek istiyorum.  
**Kabul Kriterleri:**
- Durum filtreleri (Yolda, Teslim Edildi, Geciken vb.)
- Takip numarası ile arama
- Müşteriye takip linki gönderme butonu

## 3. İade Süreçleri

**Hikaye 4:**  
Müşteri hizmetleri olarak, iade talebi gelen sipariş için kargo etiketi oluşturmak istiyorum.  
**Kabul Kriterleri:**
- İade kargosu oluşturma
- İade kargolarının ayrı listelenmesi
- İade tamamlandığında stok güncellemesi

## 4. Ayarlar ve Entegrasyon

**Hikaye 5:**  
Mağaza sahibi olarak, kendi kargo firmamı sisteme eklemek ve API bilgilerini girmek istiyorum.  
**Kabul Kriterleri:**
- Yeni kargo firması ekleme formu
- API bilgilerini girme alanı
- “Test Et” butonu ile entegrasyon testi

**Hikaye 6:**  
Yönetici olarak, hangi kargo firmalarının aktif olacağını belirlemek istiyorum.

## 5. Dashboard

**Hikaye 7:**  
Yönetici olarak, kargo operasyonlarının genel durumunu hızlıca görmek istiyorum (kaç kargo gönderildi, geciken var mı vb.)
