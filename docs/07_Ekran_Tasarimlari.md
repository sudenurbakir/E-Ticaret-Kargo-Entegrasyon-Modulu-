# 07 - Ekran Tasarımları (Mockup Listesi)

Bu doküman, kargo modülünde yer alacak ana ekranları ve içeriklerini tanımlar.

## 1. Ana Dashboard Ekranı
- Günlük kargo özeti (gönderilen, teslim edilen, geciken)
- Kargo firmalarına göre dağılım grafiği
- Geciken kargolar uyarısı
- Hızlı aksiyon butonları (Yeni Kargo Oluştur, İade Talepleri)

## 2. Siparişler Listesi Ekranı
- Tüm siparişlerin listesi
- Filtreleme (Durum, Tarih, Kargo Firması)
- Toplu "Kargoya Ver" butonu
- Her sipariş için "Kargoya Ver" butonu

## 3. Kargolarım Ekranı
- Gönderilen tüm kargoların listesi
- Filtreleme (Durum, Kargo Firması, Tarih)
- Kargo takip numarası, firma, durum bilgisi
- Tek tıkla durum güncelleme

## 4. İade Kargoları Ekranı
- Tüm iade taleplerinin listesi
- İade durumu (Bekliyor, Kargo Oluşturuldu, Depoda, Tamamlandı)
- İade kargo etiketi oluştur butonu
- Depo incelemesi ve stok güncelleme alanı

## 5. Kargo Entegrasyon Ayarları Ekranı
- Mevcut kargo firmalarının listesi (Pusula Kargo, Meltem Dağıtım vb.)
- "Yeni Kargo Firması Ekle" butonu
- API bilgilerini girme formu (Base URL, API Key, Secret vb.)
- "Test Et" butonu
- Entegrasyon durumu (Aktif / Pasif / Hata)

## 6. Kargo Detay Ekranı
- Seçilen kargonun tüm detayları
- Takip linki
- Durum geçmişi (timeline)
- Manuel durum güncelleme

## 7. Yeni Kargo Oluşturma Modalı
- Sipariş bilgisi
- Kargo firması seçimi
- Hizmet tipi seçimi
- Adres bilgisi (otomatik doldurulmalı)
- Kargo etiketi önizlemesi
