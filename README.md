# Kargo Entegrasyon Modülü

Mevcut e-ticaret admin paneline entegre edilecek kargo süreç yönetim modülü.

## Proje Hakkında

Bu proje, e-ticaret platformunda satış yapan firmaların kargo işlemlerini tek bir panel üzerinden yönetebilmesi için tasarlanmıştır. Mevcut admin paneline modüler olarak eklenebilecek şekilde planlanmıştır.

## Amaç

- Kargo süreçlerini baştan sona dijital ve merkezi hale getirmek
- Operasyonel verimliliği artırmak
- Kargo takip ve iade süreçlerini kolaylaştırmak
- Birden fazla kargo firması arasında esnek yönetim sağlamak

## Desteklenen Kargo Firmaları (Hayali)

- SwiftKargo
- Atlas Lojistik  
- Nova Express
- PowerTaşıma

## Özellikler

- Tek tıkla kargo gönderme (bireysel ve toplu)
- Otomatik kargo etiketi oluşturma
- Gerçek zamanlı kargo takibi
- İade kargo yönetimi
- Kargo maliyeti hesaplama ve firma karşılaştırması
- Kargo dashboard ve performans raporları

## Proje Dosya Yapısı

- `docs/` → Gereksinim, analiz ve dokümantasyon dosyaları
- `mockups/` → Admin paneli ekran tasarımları
- `flows/` → Süreç akış diyagramları
- `prototype/` → Basit demo arayüz
- `assets/` → Görseller

## Kullanılan Araçlar

- Draw.io (süreç akışları)
- Figma (mockup tasarımları)
- Markdown (dokümantasyon)
